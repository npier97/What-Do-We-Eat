# What Do We Eat (Still In Progress)

https://npier97.github.io/WhatDoWeEat

## Overview

**What Do We Eat** is a web application that suggests food recipes based on available ingredients in your fridge or cupboard. The goal is to help users reduce food waste and make meal planning easier. ## Features

- Input available ingredients to get recipe suggestions.
- Filter recipes by dietary preferences.
- Save favorite recipes for later use.
- Responsive UI for desktop and mobile users.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS
- **Build Tool:** Vite
- **Testing:** Vitest
- **Deployment:** GitHub Pages
